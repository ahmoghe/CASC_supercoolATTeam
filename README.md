# CASC_supercoolATTeam
All code files for the mechanisms for our product PageWave. 
# Included descriptions of the mechanisms of our assistive reading device and code along with it.

## button_trigger.py
 - Triggers the mechanism to start the page flip process
 - Controlled by a boolean when the button is clicked or not (probably).

## page_turner.py
 - MVP: Biggest selling point, by itself turns pages and sets for the next page
 - Controlled by the button mechanism
## devil_horns.py
 - Keeps the pages in place.
 - Activated to turn or rest based on page_turner.py.
